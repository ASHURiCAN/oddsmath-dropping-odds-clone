# OddsMath Dropping Odds Clone

Clone interactif de la section **Dropping Odds** d'OddsMath.com.

## Fonctionnalités
- Tableau des cotes en baisse avec mise en surbrillance
- Alertes sonores en temps réel
- Mode live / auto-refresh
- Graphiques SVG d'historique des cotes
- Filtres avancés (date, recherche, % minimum)
- Tri par colonne
- Entièrement en français

## Démarrage

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000)

## Déploiement

Ce projet est optimisé pour Vercel. Importez le repo sur [vercel.com/new](https://vercel.com/new) pour un déploiement instantané.

## Inspiration
Inspiré par [oddsmath.com](https://www.oddsmath.com/)

Clone créé avec Next.js, TypeScript et Tailwind CSS.
